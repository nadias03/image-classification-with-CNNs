# Image Classification with CNNs

## Project Overview
The goal of this project is to design, train, and evaluate convolutional neural networks (CNNs) for image classification on the CINIC-10 dataset with a significant focus on researching and experimenting with different architectures and configurations. The project aims to compare different CNN architectures and analyze their performance.

## Dataset
- **Dataset:** CINIC-10
- **Source:** [Kaggle - CINIC-10](https://www.kaggle.com/datasets/mengcius/cinic10)
- **Description:** CINIC-10 dataset is an extension of CIFAR-10 by adding reduced-resolution ImageNet images. It contains 270 000 images assigned to one of the 10 classes. The dataset is split into train (90 000 images), validation(90 000 images), and test (90 000 images) sets. Train and validation sets can be combined in order to create a bigger training set. Each class in these subsets contains 9 000 images.
